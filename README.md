# 📊 Sales & Demand Forecasting Dashboard

## Overview

This project was developed as part of the **Future Interns Machine Learning Internship Program**.

The goal of the project is to forecast future sales using historical business sales data and provide actionable insights for business planning.

The solution includes data preprocessing, feature engineering, machine learning forecasting, performance evaluation, and an interactive Streamlit dashboard for visualization.

---

## Features

* Data Cleaning and Preprocessing
* Time-Based Feature Engineering
* Monthly Sales Trend Analysis
* Machine Learning Forecasting using Random Forest Regression
* Model Evaluation using Mean Absolute Error (MAE)
* Interactive Streamlit Dashboard
* Future Sales Prediction for the Next 12 Months
* Business-Friendly Data Visualization

## Dataset

Superstore Sales Dataset

The dataset contains historical sales records including:

* Order Date
* Product Information
* Sales
* Profit
* Quantity
* Region and Category Details

---

## Project Workflow

### 1. Data Preparation

* Loaded and cleaned the sales dataset
* Handled missing values
* Converted date columns into datetime format

### 2. Feature Engineering

* Extracted Year, Month, and Quarter information
* Created time-based numerical features for forecasting

### 3. Sales Analysis

* Aggregated monthly sales data
* Visualized historical sales trends

### 4. Model Development

* Trained a Random Forest Regression model
* Generated future sales forecasts

### 5. Evaluation

* Evaluated model performance using Mean Absolute Error (MAE)

### 6. Dashboard Development

* Built an interactive dashboard using Streamlit
* Displayed forecasts, metrics, and visualizations
